## 🎯 Descripción del Proyecto

Este proyecto implementa un sistema de monitoreo de seguridad en tiempo real utilizando la plataforma **Wazuh SIEM**, con el objetivo de detectar, analizar y responder ante amenazas informáticas en una red controlada.

Se simula un entorno real con:
- Un equipo atacante
- Un equipo víctima
- Un servidor SIEM (Wazuh)
